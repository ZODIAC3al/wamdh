# Wamdh User Guide - Complete Tutorial

## Getting Started

### 1. Account Setup
1. Open Wamdh and select your role (Student/Instructor)
2. Sign up or log in with email/password or Google
3. Complete profile setup - add photo, bio (optional)
4. Choose your learning language preference

## Navigation & Core Features

### Bottom Tab Bar (5 Tabs)
```
[Home] [Notes] [AI Tutor] [Messages] [Profile]
```

**Home**: Your learning dashboard with daily tasks and announcements  
**Notes**: Your personal knowledge base with AI-powered features  
**AI Tutor**: Central sparkles button - your AI study assistant  
**Messages**: Direct and group conversations  
**Profile**: Your stats, settings, and achievement badges

### Wamdh Hub (Floating Action Button)
Tap the red "+" button to access all tools:
- Quiz Center: Take AI-generated quizzes
- Flashcards: Spaced repetition review
- Study Planner: Schedule your week
- Focus Timer: Pomodoro sessions with analytics
- Study Community: Join/create study groups
- Whiteboard: Collaborative sketching
- Analytics: Performance insights

## Feature Walkthrough

### Notes System
1. **Create a Note**: 
   - Tap "+" in Notes tab
   - Upload PDF/image or type/paste text
   - AI auto-extracts summary/key points

2. **AI Actions** (in note viewer):
   - Summarize: Condenses long notes
   - Key Points: Extracts main concepts
   - Quiz: Generates 5-question quiz
   - Flashcards: Creates study deck
   - Share: Export to other apps

### Study Community (WhatsApp-Style Groups)
1. **Join/Create Groups**:
   - From Wamdh Hub → Study Community
   - Create group with custom permissions
   - Invite classmates via link or email

2. **Group Features**:
   - Real-time chat with typing indicators
   - Sticker reactions (tap "+" to add)
   - Shared whiteboard access
   - Admin controls: lock/unlock messaging

3. **Group Settings**:
   - Toggle "Admins Only" posting
   - Switch role to test permissions
   - Mute notifications

### AI Tutor
1. **Ask Questions**: Type naturally, get instant answers
2. **Voice Chat**: Tap mic icon for spoken conversation
3. **Context Mode**: Open from note to discuss that content
4. **Quiz Generation**: "Quiz me on photosynthesis"

### Focus Timer (Pomodoro)
1. **Choose Session Type**: Focus (25m), Long (50m), Short (15m)
2. **Start Timer**: Large centered play button
3. **Track Sessions**: Tomato counter increases automatically
4. **Weekly Stats**: Minutes accumulate toward goal (4hr default)

### Gamification
- **Daily Streak**: +50 XP for completing all tasks
- **Achievement Badges**: Earn for milestones
- **Level Progression**: Based on XP points
- **Leaderboard**: Weekly ranking among peers

## For Instructors

### Course Management
1. **Create Course**: Wamdh Hub → Exam Creator
2. **Add Chapters**: Lock/unlock for student access
3. **Generate Quizzes**: AI creates questions from notes
4. **View Analytics**: Drop-off points, avg scores

### Student Interaction
1. **Announcements**: One-to-many messaging
2. **Office Hours**: Schedule in Messages
3. **Assignment Grading**: AI-assisted grading

## Advanced Tips

### Keyboard Shortcuts (Web)
- `Ctrl+N`: New note
- `Ctrl+Shift+T`: Focus Timer
- `Ctrl+K`: Quick search notes
- `/command`: Command palette

### Offline Mode
- Notes download for offline access
- Syncs changes when online
- Draft autosave locally

### Customization
- Theme: Light/Dark/System
- Study hours goal: Adjust in Profile
- Notification schedule: Settings menu

## Troubleshooting

**Notes not loading?**
- Check internet connection
- Pull to refresh

**AI response slow?**
- Complex queries take longer
- Check backend status in Settings

**Timer not working?**
- Ensure notification permissions enabled

**Share not working?**
- Install sharing app (native feature)
